Projekt: Responsive Taschenrechner-Webanwendung

Entwicklung eines voll responsiven Taschenrechners, der sich automatisch an verschiedene Smartphone- und Bildschirmformate anpasst und vollständig ohne Scrollen nutzbar ist. Die Anwendung wurde speziell für eine optimale mobile Benutzererfahrung konzipiert.

Der Taschenrechner verfügt über eine präzise Rechenfunktion mit einer Rundung auf drei Nachkommastellen. Eingaben werden visuell hervorgehoben und zweifarbig dargestellt, um die Benutzerfreundlichkeit und Lesbarkeit zu verbessern.

Zusätzlich wurde eine Funktion implementiert, die es ermöglicht, die Anwendung als Icon direkt auf dem Home-Bildschirm zu speichern (Progressive-Web-App-ähnliches Verhalten), wodurch ein schneller Zugriff gewährleistet wird.

Im Laufe der Entwicklung wurden mehrere technische Fehler identifiziert und behoben, was zu einer stabileren und verbesserten Gesamtanwendung führte. Durch das Projekt konnten umfassende Kenntnisse in Webentwicklung, responsivem Design und Fehleranalyse vertieft werden.
